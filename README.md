# 📦 Slider Fragment for Liferay

## 🧩 About

The **Slider Fragment** is a customizable carousel component built for Liferay using HTML, JavaScript, and CSS.

It was originally developed during a hackathon with a focus on:

* Performance
* Ease of use in Liferay CMS
* Flexibility for different content types and styles within Liferay

---

## ✨ Features

* 📸 Multiple slides support
* 📝 Editable content (title, description, button)
* 🎯 Navigation controls (arrows and indicators)
* ⚡ Lightweight (no external libraries required)
* 🎨 Customizable styling via Liferay configurations
* 🧩 Native tagging support built-in

## 🚀 How to Use

1. Go to **Liferay → Fragments**

2. Create a new fragment

3. Copy and paste the files:

   * HTML
   * CSS
   * JavaScript

4. Configure editable fields using `data-lfr-editable`

5. Publish and use it on your page

---

## 🧠 Concepts Used

* Editing text and images using Liferay’s `data-lfr-editable` attribute
* Creating configurations to enable no-code customization of the fragment
* Image optimization and rendering focused on performance
